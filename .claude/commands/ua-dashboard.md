# /ua-dashboard — Interactive Knowledge Graph Dashboard

Launches the Understand-Anything interactive dashboard for visual
exploration of the knowledge graph.

Features:
- Force-directed graph layout
- Semantic search
- Guided architectural tours
- Filter by file, function, class, domain

## Usage

```
/ua-dashboard
```

Opens at `http://localhost:5174`.

Requires a knowledge graph (run `/ua-analyze` first).
