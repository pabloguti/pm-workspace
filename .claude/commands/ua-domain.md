# /ua-domain — Domain Concept Extraction

Extracts business domain entities, processes, and flows from a codebase
using Understand-Anything's domain graph pipeline.

## Usage

```
/ua-domain ~/claude
/ua-domain ~/projects/foo
```

Results feed into the knowledge graph as `DOMAIN_ENTITY` nodes.
