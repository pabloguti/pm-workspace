# /ua-onboard — Codebase Onboarding Guide

Generates a guided tour of a codebase's architecture, key files,
and dependency structure for new developers or agents.

## Usage

```
/ua-onboard ~/claude
/ua-onboard ~/projects/foo
```

Integrates with Savia's project-onboarding skill for contextual tours.
