## Instalación

Para instalar un MCP de cualquier categoría, usa:

```bash
npx claude-code-templates@latest --mcp {category}/{name} --yes
```

**Ejemplo:**
```bash
npx claude-code-templates@latest --mcp database/postgresql-schema-inspector --yes
```
