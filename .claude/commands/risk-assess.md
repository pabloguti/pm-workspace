# Command: /risk-assess

Calculate risk score for a task and display comprehensive breakdown with review level recommendation.

## Usage

```
/risk-assess {task-id}
/risk-assess {PBI-id}
```

## Parameters

- `$ARGUMENTS`: Task ID or PBI ID to assess

## Output

Risk score calculation with breakdown and review level recommendation.
