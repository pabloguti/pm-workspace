# /ua-chat — Semantic Knowledge Graph Search

Queries the knowledge graph using natural language. Searches across
files, functions, classes, dependencies, and domain concepts.

## Usage

```
/ua-chat what commands manage sprints?
/ua-chat which skills depend on memory-agent?
/ua-chat how does CI validation work?
```

Requires a knowledge graph to exist (run `/ua-analyze` first).
