# Domain: Evaluations Framework

## Contexto

El framework de evaluaciones proporciona un sistema estructurado para medir la calidad de outputs de agentes de software, desde descomposición de requisitos hasta precisión de estimaciones.

## Conceptos Clave

- **Evaluation Set**: Conjunto de items a evaluar (PBIs, specs, reviews, etc.)
- **Rubric**: Escala de puntuación con criterios claros (90-100: Excellent, 70-89: Good, etc.)
- **Trend Analysis**: Análisis de patrones en múltiples evaluaciones
- **Regression Detection**: Alertas cuando la calidad cae más del 10%

## Relacionado Con

- Sprint management y planning
- PBI refinement y estimation
- Code review y quality assurance
- Agent skill improvement
