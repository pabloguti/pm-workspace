# Command Family Icons

## Objetivo

Update and modernize command icons in savia-mobile. Each command family and individual command should have a distinct, recognizable icon.

## Requisitos

- Use Material Symbols (already in the project) for consistency
- Each of the 10 command families gets a unique icon
- Individual commands within a family inherit the family icon or get a specific one
- Icons should be meaningful (not decorative)
- Support dark/light theme variants
