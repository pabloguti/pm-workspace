# Vault Template

Este es el template canonico de un vault Obsidian por proyecto.

El comando `/vault-init {slug}` copia esta estructura a `projects/{slug}/vault/`
y substituye los placeholders `{{PROJECT_NAME}}`, `{{PROJECT_SLUG}}`, `{{CURRENT_SPRINT}}`.

Ver: `docs/specs/SPEC-128-obsidian-context-as-code.spec.md`
