## Descripcion

As an international user I want the application to support English and Spanish so that I can use it in my preferred language. Use Angular i18n with ngx-translate. All user-facing strings must be externalized to JSON translation files.

## Criterios de Aceptacion

- [ ] Language selector in header with ES/EN options
- [ ] All UI strings loaded from assets/i18n/{lang}.json
- [ ] Date, number, and currency formats adapt to locale
- [ ] Backend validation messages returned with Accept-Language
- [ ] Default language detected from browser preference
