## Proyectos open source

- **[claude-code-templates](https://github.com/davila7/claude-code-templates)** — Daniel Avila (@davila7). El mayor marketplace de componentes para Claude Code (5.788+ componentes, 21K+ stars). Base estructural del sistema de comandos y skills.

- **[kimun](https://github.com/lnds/kimun)** — Eduardo Díaz (@lnds). Inspiró las scoring curves: normalización piecewise linear para métricas de calidad (PR size, context usage, file size, velocity deviation, test coverage, Brier score).

- **Engram** (Anthropic). Inspiró el sistema de memoria persistente JSONL con deduplicación, compresión semántica y grafo de conocimiento.

- **claude-mem**. Inspiró mejoras de memoria y resolución de comandos en lenguaje natural.

- **BullshitBench**. Inspiró el Quality Validation Framework: consenso multi-juez, calibración de confianza y validación de coherencia de output.

## Estudios e investigación

- **LLYC — "Espejismo de Igualdad" (2026)**. Base del equality-shield: detección y bloqueo de 6 tipos de sesgo de género en outputs generados por IA.

- **Miguel Luengo-Oroz — AI Augmented Donut**. Análisis de oportunidades de aumentación por IA por sector. Referencia para `/ai-exposure-audit` y la guía de oportunidades de aumentación.

- **DX Core 4**. Framework de Developer Experience que fundamenta `/dx-survey` y `/dx-dashboard`: métricas de satisfacción, eficiencia, flujo e impacto.

- **NIST AI RMF, ISO/IEC 42001, EU AI Act**. Frameworks de gobernanza de IA que fundamentan `/audit-trail`, `/governance-check` y el sistema de compliance.

- **Fundación ONCE — "Por Talento Digital"**. Referencia de inclusión digital de personas con discapacidad (30.000+ formados). Base para el sistema de accesibilidad universal.

- **N-CAPS (Nonlinear Context-Aware Prompting System)**. Sistema de prompts adaptativos para trabajadores con discapacidad cognitiva. Inspiró `/guided-work` y el protocolo de trabajo guiado.

- **arxiv 2411.13950 — ADHD/Autism in Software Development**. Evidencia de TDD como accesibilidad, rejection sensitivity en code reviews, task decomposition como soporte cognitivo.

- **arxiv 2507.06864 — ADHD-Aware Productivity Framework**. Body doubling digital, DopBoost micro-interventions, detección de fatiga. Inspiró el diseño de `/guided-work` y `/focus-mode`.

- **O\*NET** (U.S. Department of Labor). Datos ocupacionales para el análisis de exposición laboral a IA en `/ai-exposure-audit`.
